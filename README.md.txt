# San Valentín

Este es un proyecto web interactivo diseñado para San Valentín. Contiene una pregunta con opciones "Sí" y "No" para interactuar con el usuario. Dependiendo de las elecciones del usuario, la página responde dinámicamente.

## Características
- Si el usuario selecciona "No", el botón de "Sí" crece progresivamente en tamaño.
- Si el usuario selecciona "Sí", se muestra una carta con un mensaje especial.
- Fondo decorativo de rosas rojas para complementar el tema de San Valentín.

## Cómo usar
1. Clona este repositorio:
   ```bash
   git clone https://github.com/tu-usuario/san-valentin.git
   ```
2. Abre el archivo `index.html` en tu navegador.

## Subir a GitHub Pages
1. Asegúrate de que el repositorio esté en tu cuenta de GitHub.
2. Ve a **Settings** > **Pages**.
3. En la sección **Branch**, selecciona `main` y carpeta raíz (`/`).
4. Obtén tu enlace para compartir, que tendrá un formato como: `https://tu-usuario.github.io/san-valentin/`.
